Radio Havana Cuba

Radio Havana Cuba broadcasts to North America on 6000 KHz evenings with a one hour program that is repeated throughout the evening.

http://radiohc.cu/en

Listen on shortwave via NO2CW Hollywood (Miami) FL, 242 miles northeast of Havana.

http://qth.ddns.net:8073/?f=6000.00amz10
